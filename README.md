# CIRCUITPY-Workshop-Facilitator
Repo for workshop facilitator in case their device gets corrupted. Contains all working code from basic project 0, Meatball project 1, BME680 project 2, APDS project 3, Snake game project 4, Donkey Kong project 5, Super Mario Bros project 6. Also contains all required libraries, sprites, BMP, WAV files, as well as PDF schematics.
